# clr
Clear your terminal 3x

Usage
---
Default clear by 200 lines
```ssh
$ clr
```

Clear exactly 10 lines
```sh
$ clr 10
```

Clear plus 50 lines (200 + 50 lines)
```sh
$ clr +50
```

Clear minus 50 lines (200 - 50 lines)
```sh
$ clr -50
```

Installation
---
Build the tool
```sh
go get https://github.com/maiah/clr
go install
```
